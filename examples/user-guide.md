# Photo Editor Web App — Руководство пользователя

## Введение

**Photo Editor Web App** — это простое веб-приложение для редактирования изображений прямо в браузере. Вы можете загружать фото, применять фильтры, обрезать, изменять размер и сохранять результат.

## Системные требования

- Браузер: Chrome, Firefox, Safari, Edge (последние 2 версии).
- Интернет: обычное подключение.
- Хранилище: минимум 50 МБ свободного места для кэша браузера.

## Начало работы

### Шаг 1: Открыть приложение

1. Перейдите на `https://photoeditor.example.com`.
2. Дождитесь загрузки интерфейса (обычно 2–3 секунды).
3. Вы увидите главный экран с кнопкой **"Загрузить фото"**.

### Шаг 2: Загрузить фотографию

1. Нажмите кнопку **"Загрузить фото"** в центре экрана.
2. Выберите файл с вашего компьютера (поддерживаются JPG, PNG, GIF, WebP).
3. Максимальный размер файла — 20 МБ.
4. После выбора фото появится на рабочем полотне.

### Шаг 3: Применить фильтры

На панели слева найдёте секцию **"Фильтры"**:

| Фильтр | Описание | Диапазон |
|--------|---------|----------|
| Яркость | Увеличивает или уменьшает яркость | -100 до +100 |
| Контраст | Усиливает контраст | -100 до +100 |
| Насыщенность | Управляет насыщенностью цветов | -100 до +100 |
| Размытие | Применяет размытие по Гауссу | 0–20 пикселей |
| Сепия | Придаёт коричневый оттенок | Вкл/Выкл |
| Чёрно-белое | Убирает цвета | Вкл/Выкл |

**Пример:** чтобы сделать фото ярче:
1. Найдите ползунок "Яркость".
2. Перетащите его вправо (значения 0–100).
3. Изменения видны в реальном времени.

### Шаг 4: Обрезать или изменить размер

#### Обрезка (Crop)

1. Нажмите кнопку **"Обрезать"** на панели инструментов.
2. На фото появится прямоугольник с углами для изменения размера.
3. Перетащите углы, чтобы выбрать нужную область.
4. Нажмите **"Применить"** или **"Отмена"**.

#### Изменение размера

1. Откройте **"Размер изображения"** в меню.
2. Введите новые значения ширины и высоты (в пикселях).
3. Отметьте **"Сохранить соотношение сторон"**, если нужно.
4. Нажмите **"Применить"**.

### Шаг 5: Сохранить результат

1. Нажмите кнопку **"Скачать"** в верхнем правом углу.
2. Выберите формат файла (JPG, PNG, WebP).
3. Выберите качество (низкое, среднее, высокое).
4. Нажмите **"Скачать"** — файл сохранится на ваш компьютер.

## Горячие клавиши

| Клавиша | Действие |
|---------|----------|
| Ctrl+Z | Отменить последнее действие |
| Ctrl+Y | Повторить отменённое действие |
| Ctrl+S | Открыть диалог сохранения |
| R | Открыть инструмент обрезки |
| F | Развернуть на весь экран |

## Часто задаваемые вопросы

**В: Почему фото не загружается?**  
О: Проверьте:
- Формат файла (JPG, PNG, GIF, WebP).
- Размер файла (не более 20 МБ).
- Соединение с интернетом.
- Браузер обновлён до последней версии.

**В: Можно ли отменить изменения?**  
О: Да, нажмите Ctrl+Z или кнопку "Отменить" на панели инструментов. История сохраняется для текущей сессии.

**В: Какой формат выбрать при сохранении?**  
О: 
- **JPG** — для фотографий (меньший размер файла).
- **PNG** — если нужна прозрачность.
- **WebP** — современный формат с лучшей компрессией.

**В: Теряется ли качество при применении фильтров?**  
О: Нет, фильтры — неразрушающие. Можно отменить любой фильтр в любой момент.
